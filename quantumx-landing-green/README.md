# QuantumX - ApexQuant Landing Page

Questa landing page è stata sviluppata per **Antonio Vida** seguendo lo stile **Sartori** (Direct Response Marketing) e utilizzando il tema **ApexQuant Dark Mode**.

## Tecnologie Utilizzate

- **React + Vite**: Framework principale per velocità e modularità.
- **Tailwind CSS v4**: Styling utility-first con tema personalizzato (Dark Blue + Gold).
- **GSAP**: Animazioni scroll-based per Hero, Features, About.
- **Framer Motion**: Animazioni UI per Testimonials e FAQ.
- **React Hook Form**: Gestione del form di contatto.
- **Lucide React**: Icone moderne e leggere.

## Struttura del Progetto

- `src/components/Hero.jsx`: Sezione principale con hook forte e CTA.
- `src/components/Problems.jsx`: Agitazione del problema (Trading tradizionale vs Metodo Magnitudo).
- `src/components/About.jsx`: Presentazione di Antonio Vida e della soluzione.
- `src/components/Features.jsx`: I 3 pilastri del sistema ApexQuant.
- `src/components/Testimonials.jsx`: Riprova sociale.
- `src/components/FAQ.jsx`: Gestione obiezioni.
- `src/components/CTA.jsx`: Modulo di contatto finale.
- `src/components/Footer.jsx`: Note legali e copyright.

## Installazione e Avvio

1. Installare le dipendenze:
   ```bash
   npm install
   ```

2. Avviare il server di sviluppo:
   ```bash
   npm run dev
   ```

3. Costruire per la produzione:
   ```bash
   npm run build
   ```

## Note sullo Stile

Il design segue rigorosamente la palette:
- **Background**: `#010B32` (Deep Navy)
- **Card Background**: `#1A2346` (Lighter Navy)
- **Accent**: `#FBC737` (Gold/Amber)
- **Text**: `#BCD2EE` (Soft Blue-White)
- **Urgency**: `#AF0505` (Red)

L'approccio copy è diretto, focalizzato sul dolore del "trading emotivo" e sulla soluzione "matematica/istituzionale".
